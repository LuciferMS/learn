单例设计模式
    1.要点
        1.1 某个类只能有一个实例
                构造器私有化，不对外公开实例化逻辑
        1.2 必须由这个类自行创建这个实例
                含有一个该类的静态内部变量来保存这个唯一的实例
        1.3 它必须自行向整个系统提供这个实例
                对外提供给获取该实例的方式
    2.常见形式
        2.1 饿汉式：直接创建实例，不存在线程安全问题
               2.1.1 直接实例化
               2.1.2 枚举
               2.1.3 静态代码块
        2.2 懒汉式: 延迟对象的创建
                2.2.1 线程不安全
                2.2.2 线程安全
                2.2.3 静态内部类
